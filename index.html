This Orb was found with along with a package called "Creative Cloud." Currently being used to create vector graphics and large print desings.

Orb of InDesign

The Orb was used first in a just couple college classes. I later found this Orb becoming a staple for creating any print project.

One of the most commonly used potions in all the land. It’s very refined and tasty. Not sure where it gets its blue color from... maybe a reflection from the sky.

The Lens of Discovery is used to find the root of a problem and helps perform research and discovery. Using it correctly will provide accurate reference material, documentation, and inspiration.

Scope Spyglass

Heavily used with the Strategy Brush and Lens of Discovery, the Scope Spyglass helps define hard objectives like time and deliverables. I stole it from pirates who used it to steal treasure.
